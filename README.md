#  Coffee Sales Excel Dashboard

##  Project Overview
This project analyzes coffee shop transaction data and presents insights through an interactive Excel dashboard.  
It demonstrates advanced Excel skills including:
- Pivot tables and slicers for dynamic filtering
- KPI cards for headline metrics
- Charts for weekday, hourly, and location‑based trends
- Dashboard design and layout formatting

## 📂 Files in this Repository
- `coffee_sales_dashboard.xlsx` → Main Excel workbook containing raw data and dashboard
- `coffee_sales_raw.csv` → Raw transaction dataset (optional)
- `README.md` → Documentation and usage guide
- `screenshots/` → Dashboard and chart previews

## ⚙️ How to Use
1. Download `coffee_sales_dashboard.xlsx`.
2. Open in Excel (2016 or later).
3. Use slicers to filter by:
   - Day Name  
   - Transaction Date  
   - Product Category  
4. Explore charts for:
   - Sales count variation by weekday and hour  
   - Revenue comparison across store locations  
   - Product‑wise sales and revenue
